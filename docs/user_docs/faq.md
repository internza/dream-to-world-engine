# FAQ

## What does this project do
It converts a dream description into a structured world model in JSON.

## Do I need special software
You need Node.js and npm.

## Can I change the dream input
Yes. Change the hardcoded dream string in the source code, then run npm start.

## Is the output always the same
Yes, for the same input, the current version is deterministic.

## Does it have a UI
Not in the current version.
